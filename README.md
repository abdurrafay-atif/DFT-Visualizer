This project is a WIP and the end-result will be a home-made version of the Discrete Fourier Transform (DFT) implemented in the C Programming Language that uses the standard matrix multiplication algorithm, as well as with the Fast Fourier Transform (FFT) algorithm. This project will also include many other features that may prove useful to some interested in signal processing in the C language.

This project is mainly for me to learn some more advanced C skills and learn about how to handle larger C projects.

What Is Included:
* Library for Complex Numbers + Arithmetic
* Library for Discrete, Finite Sinusoids
* Library for Matrix / Vector Operations + Vector Transformations
* Library for the DFT / FFT Algorithms
